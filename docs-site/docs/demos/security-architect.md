# Security Architect Demo

## Goal

Show how CyberArmor.AI connects services, control points, telemetry, policies,
and evidence into an operating architecture.

## Seed

```bash
bash scripts/demo/run_security_architect_demo.sh
```

## Flow

1. Open Admin Portal > Overview.
2. Show tenant readiness score.
3. Open Policies and Policy Studio.
4. Open SDK & Onboarding to show bootstrap and integration paths.
5. Open Reports & Evidence Export and download the full pack.

## Architecture points

- control plane
- policy service
- endpoint telemetry
- AI router/provider context
- audit/evidence records
- tenant-scoped customer portal

## Close

CyberArmor.AI is not a standalone prompt filter. It is a control and evidence
layer that fits into enterprise security operations.
