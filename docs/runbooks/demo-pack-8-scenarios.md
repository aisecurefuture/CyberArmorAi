# Demo Pack: 8 Scenarios

Run all:

```bash
bash scripts/demo/run_all_8_demos.sh
```

Run individually:

1. Attack to Action
```bash
bash scripts/demo/run_demo_1_attack_to_action.sh
```

1. Human-in-the-Loop Approval
```bash
bash scripts/demo/run_demo_2_human_approval.sh
```

1. Delegation Chain
```bash
bash scripts/demo/run_demo_3_delegation_chain.sh
```

1. Tenant Isolation
```bash
bash scripts/demo/run_demo_4_tenant_isolation.sh
```

1. Provider Governance
```bash
bash scripts/demo/run_demo_5_provider_governance.sh
```

1. Incident Triage
```bash
bash scripts/demo/run_demo_6_incident_triage.sh
```

1. Key Rotation + Blast Radius
