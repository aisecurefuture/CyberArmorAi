Installation complete

The CyberArmor Endpoint Agent is now running as the ai.cyberarmor.endpoint LaunchDaemon.

Verify

sudo launchctl list | grep cyberarmor
sudo tail -n 30 /var/log/cyberarmor/agent.log

Finish bootstrap (only if no BOOTSTRAP_TOKEN was provided)

If /etc/cyberarmor/agent.json shows an empty api_key, redeem a one-time bootstrap token from the customer portal:

sudo /usr/local/cyberarmor/.venv/bin/python3 \
  /usr/local/cyberarmor/installer.py install \
  --control-plane-url https://app.cyberarmor.ai \
  --bootstrap-token YOUR_TOKEN

Clipboard helper

The per-user clipboard helper was auto-installed for the logged-in user (default mode: monitor). To switch enforcement modes later:

sed -i '' 's/"clipboard_action": "[^"]*"/"clipboard_action": "redact"/' \
  ~/.config/cyberarmor/helper.json
launchctl kickstart -k "gui/$(id -u)/ai.cyberarmor.clipboard-helper"

Other users on the same machine should run, once after login:

/usr/local/cyberarmor/install_clipboard_helper.sh

Uninstall

sudo /usr/local/cyberarmor/.venv/bin/python3 \
  /usr/local/cyberarmor/installer.py uninstall
/usr/local/cyberarmor/uninstall_clipboard_helper.sh
