CyberArmor Endpoint Agent

This installer will:

  • Install the endpoint agent to /usr/local/cyberarmor/
  • Register the system service (ai.cyberarmor.endpoint LaunchDaemon)
  • Configure the per-user clipboard helper for the active console user

If /etc/cyberarmor/install.env exists with a BOOTSTRAP_TOKEN, the agent will redeem it during install and be ready immediately. Otherwise you'll need to finish the bootstrap manually after install.

Administrator privileges are required.
